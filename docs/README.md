# GitHub Pages Documentation

This directory contains the GitHub Pages site for grakchawwaa-bot.

## Setup

1. Go to your repository settings on GitHub
2. Navigate to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Select **main** (or your default branch) and **/docs** folder
5. Click **Save**

Your Terms of Service will be available at:
`https://[your-username].github.io/grakchawwaa-bot/terms-of-service.html`

## Files

- `terms-of-service.md` - The Terms of Service document (Markdown)
- `_config.yml` - Jekyll configuration for GitHub Pages
- `index.md` - Landing page with link to Terms of Service

## Updating

Simply edit `terms-of-service.md` and push to your repository. GitHub Pages will automatically rebuild the site.
